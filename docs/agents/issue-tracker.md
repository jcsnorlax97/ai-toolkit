# Issue Tracker

## Default

This repo uses local markdown issues under `.scratch/issues/`.

## File Naming

- `NNNN-short-kebab-title.md`
- Example: `0001-bootstrap-typescript-cli.md`

## Required Sections

- Title
- Problem
- Desired behavior
- Scope
- Acceptance checks
- Out of scope

## Notes

- Prefer one independently shippable slice per issue.
- Split by behavior, not by frontend/backend/database layers.
- If a task cannot be validated independently, it is probably too large.
